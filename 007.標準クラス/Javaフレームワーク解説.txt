軽量、迅速系のフレームワークと、
大規模、拡張性系のフレームワークに大別されている




17年注目のフレームワーク10選より

Spring Framework
・他言語ではメジャーなフルスタックフレームワークに近い
・DIコンテナを活用して外部のフレームワークを統合することもできる
（導入方法）
１「Eclipse IDE for Java EE Developers」をダウンロードして解凍
２　EclipseメニューからHelp > Eclipse Marketplaceの順で選択、stsを入力して検索、Spring Tool Suiteのところの「install」ボタンをポチ


Struts
概要
・2000年からあるフレームワーク、古参。
・世界で最も有名なJavaフレームワークらしい
・MVC採用
・脆弱性が問題で、近年は他のフレームワークへの移行も少なくない
